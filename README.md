# Proyecto Donaton - "Ayuda en camino" (en desarrollo)
Donaton es la nueva aplicación de ayuda humanitaria en emergencias, desastres naturales y crisis sociales, enfocado en coordinar y monitorear diferentes áreas de suma importancia como la logística, inventario,
manejo de donaciones, entre otros, reemplazando totalmente el uso manual y anticuado del sistema actual.<br>
<br>
Donaton nació para mejorar el sistema actual permitiendo una mayor robustez y óptimo sistema que evitará completamente los problemas en los sistemas anticuados. <br>
Junto a una arquitectura de microservicios muy sofisticada permitiendo el mantenimiento y constante actualización de los diferentes servicios para garantizar una continúa mejora y novedades dentro de la plataforma.<br>

## Herramientas y tecnología
- Springboot
- React
- MongoDB
- vercel
- render

## Microservicios
- usuarios
- inventario
- logistica
- necesidades
- donaciones
